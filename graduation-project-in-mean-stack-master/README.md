graduation-project
==================
##A Rental Website
##Install
 - Clone / Download
 - Change to project's root directory
 

> `npm install`

> `bower install`

  - install [mongodb](http://www.mongodb.org/downloads)
  
##Run
- Change to mongodb directory(if windows)

> `mongod -dbpath data`

 - Change to project's root directory
 
> `gulp`

 - Same place
 
> `nodemon`

 

- Open [localhost:3000](localhost:3000)
